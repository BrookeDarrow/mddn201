Energy Efficieny in Web Design

How does this website run more efficiently than others?

This website maintains efficiency as a low-carbon design through several different aspects.

Firstly, this website is static. Instead of requiring a new generation every time someone visits, it is always there. A static website is one which is generated once and subsequently exists as a simple set of documents on the server's hard disk.

This website also utilises image dithering, a process which makes image files smaller by utilising a colour compression technique. Instead of making images smaller or lower quality, dithering maintains an image's physical qualities by limiting the amount of colours in an image and regenerating it. Instead of using full-colour high-resolution images, we can utilise dithering to create smaller-sized websites, supporting faster load times and energy efficiency.

Web analysis software, such as Google Analytics, keeps records of everything that happens on a website; popular pages, visitor origins, etc. This constant exchange and storage of user data, however, generates extra traffic and thus more taxing on energy consumption. This website is self-hosted, meaning that it utilises a private web server as opposed to one outside of someone's own control.
